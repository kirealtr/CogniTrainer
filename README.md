# CogniTrainer
Веб-приложение в рамках проекта к курсу "Python для решения прикладных задач". Суть в обучении статистической интуиции и борьбе с когнитивными искажениями. Проект вдохновлен книгой Даниэля Канемана "Думай медленно... Решай быстро"."

## Описание проекта

`CogniTrainer` — это учебный веб-сервис, который помогает пользователю:

- изучать когнитивные искажения;
- читать объяснения и примеры;
- запоминать идеи через карточки;
- проходить квизы;
- решать быстрые игры на выбор между рискованными и гарантированными вариантами;
- замечать, как интуитивные решения могут противоречить математическому ожиданию.

## Где менять содержимое проекта

### Если нужно изменить карточку:
1. открыть data/biases.csv;
2. найти нужную строку;
3. отредактировать название, объяснение, пример или совет.
### Если нужно изменить квиз:
1. открыть proj_maths/biases_work.py;
2. найти функцию get_quiz_questions();
3. изменить нужный вопрос в списке.
### Если нужно изменить игру:
1. открыть proj_maths/biases_work.py;
2. найти функцию get_risk_games();
3. изменить нужную игру в списке.
### Если нужно изменить страницу:
1. открыть папку templates/;
2. выбрать нужный HTML-файл;
3. изменить текст или разметку.
### Если нужно изменить внешний вид:
1. открыть static/style.css;
2. изменить нужные CSS-правила.
